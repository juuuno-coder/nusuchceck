description: 누수체크 브랜드 컬러 - primary는 밝은 파랑, teal은 점검/문제 상태용
type: project
---

primary = 밝은 파랑 (blue 계열, `#3b82f6` = primary-500)
- 브랜드 포인트 컬러, CTA 버튼, 네비게이션, 강조 요소 전반

teal = 점검 중 / 문제 상태 전용
- 청록/민트 색상은 "녹물 느낌"이 난다는 이유로 메인에서 제외
- 점검 전, 문제 감지, 경고 상태 등에만 제한적으로 사용

**Why:** 사용자가 청록/민트(기존 primary)가 "녹물같다"는 느낌이 든다고 판단해 변경
**How to apply:** 새 UI 요소는 항상 `primary-*`(파랑)을 포인트 컬러로 사용. teal은 상태 표현 전용으로만 사용할 것
