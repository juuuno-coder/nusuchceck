name: Writing style preference
description: User wants Toss-style 에요체 throughout the app, not 입니다체
type: feedback
---

모든 UI 텍스트는 **에요/이에요** 체로 작성할 것.

**Why:** 토스처럼 친근하고 가벼운 브랜드 보이스를 원함. "입니다/습니다" 체는 딱딱하게 느껴짐.

**How to apply:**
- "~합니다" → "~해요"
- "~됩니다" → "~돼요"
- "~입니다" → "~이에요" / "~예요"
- "~습니다" → "~어요/아요"
- 설명이 길어질 때는 1문장씩 끊어서 작성
- 새 페이지/텍스트 작성 시 항상 에요체 유지
