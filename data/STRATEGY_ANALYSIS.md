## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: 21개 모델, 23개 DB 테이블, Rails 7.2 풀스택
- **기술 스택**: Hotwire (Turbo+Stimulus), Tailwind CSS, ActionCable (실시간), Sidekiq 백그라운드
- **주요 기능**:
  - 고객: AI 누수 감지 빠른 점검, 전문가 매칭, 견적 관리, 보험청구 자동화
  - 전문가(마스터): 영역 기반 매칭, 수락/거절, 실시간 채팅, 정산
- **결제**: Toss Payments (에스크로, 수수료 관리)
- **배포**: NCP Docker (포트 4080), PostgreSQL + Redis + Sidekiq
- **운영 문화**: OKR 기반 개발, 일주일마다 체크인

### 매출/사용자 현황 (추정)
- **상태**: 초기 성장 단계 (NCP 배포 완료, 2026-03-24)
- **대상 시장**:
  - 고객(Customer): 주택 누수 문제 있는 사람들 (월 1,000명 추정)
  - 전문가(Master): 배관 기술자, 누수 전문가 (월 50-100명 추정)
- **비즈니스 모델**: 2-sided marketplace (에스크로 결제 기반, 수수료 + 서비스 수료)

### 핵심 과제
1. **공급-수요 밸런스**: 초기에 "공급 추월" (전문가 많아도 고객 부족) 또는 "수요 추월" (고객 대기) 가능
2. **품질 보증**: 부실 작업 발생 시 에스크로 분쟁 증가 → 마켓플레이스 신뢰도 하락
3. **지역 기반 매칭의 복잡성**: 전국 1,000+ 지역 매칭 알고리즘 고도화 필요
4. **마케팅 채널**: 누수는 응급 상황 → Google Search/Naver Knowledge만이 주요 채널
5. **단위 경제**: 고객당 평균 거래액 $80,000 (누수 수리비)이지만, 마켓플레이스 수수료 5%만 가능
6. **규제**: 시공업 등록, 보험, PCI-DSS (결제 안전)

## 3. Beachhead 분석

### Primary Segment: 서울/경기 아파트 단지 주민 (주택 누수 긴급)

**4가지 선정 기준:**

1. **Segment Size**: 서울/경기 아파트 약 300만 가구 × 연 누수 발생율 5% = 15만 명/년
2. **Execution Advantage**:
   - 긴급성(응급실) → 가격 민감도 낮음
   - AI 진단으로 기술력 입증 가능
   - 투명한 견적(표준화)로 사기 방지 신뢰 확보
   - 한국 건설 이해도 높음 (아파트 구조)
3. **Size of Beachhead**: 월 200-300건 누수 접수 가능
4. **Growth Path**: 아파트 → 단독주택/빌라 → 상가/건설사 정기점검

### Go-to-Market (Priority Order)

1. **Google Search Ads** (비용 높지만 immediate ROI)
   - "누수 전문가", "누수 응급" 키워드
   - 예상 CPC: 3,000원, CPA: 20,000원 (고객당)

2. **Naver Knowledge 파트너십**
   - "누수 어떻게 해야 하나요?" 질문에 자동 답변 + 매칭

3. **입소문 (Referral)**
   - 고객이 친구/가족 추천 (건설 비용이 크므로 신뢰 중요)

## 4. Growth Loop

### Primary Loop: **고객 신청→AI 진단→전문가 매칭→완료→신뢰 구축→입소문**

```
누수 신고 접수
  ↓
AI 빠른 점검 (1시간)
  ↓
전문가 자동 매칭 (영역 기반)
  ↓
견적 제시 (표준화)
  ↓
에스크로 결제 (안심)
  ↓
작업 완료 & 사진/영상
  ↓
별점 평가 & 리뷰
  ↓
입소문 (친구 추천)
```

**Loop 시간**: 3-4일 (신고→완료)

### Secondary Loop: **전문가 매칭 성공률 높음→재계약율 증가→경쟁력 강화**

- 매칭 성공률 95% → 전문가당 월 15-20건 안정적 수입
- 안정적 수입 → 더 많은 전문가 유입
- 많은 전문가 → 매칭 대기 시간 단축 → 고객 만족도 증가

## 5. 가격 전략

### 고객 수수료 (Commission)

**기본 구조**: 누수 수리 비용의 **5% (마켓플레이스 안전 수수료)**

| 수리 비용 | 수수료 | 실제 예시 |
|---------|--------|---------|
| 50,000원 | 2,500원 | 화장실 불렛 교체 |
| 200,000원 | 10,000원 | 파이프 누수 보수 |
| 500,000원 | 25,000원 | 대규모 누수 수리 |
| 1,000,000원 | 50,000원 | 벽체 해체 수리 |

**마진**: 고객 5% + 전문가 수수료(거래액의 5%)차감 = 마켓플레이스 순이익 10-15%

### 전문가 구독 (선택사항)

| 플랜 | 월가격 | 혜택 | 타겟 |
|------|--------|------|------|
| **Free** | 0원 | 기본 매칭, 수익 94% (수수료 6%) | 개인 기술자 |
| **Pro** | 9,900원 | 우선 매칭(1시간 내), 수익 97%, 광고 없음 | 활성 전문가(월 10건+) |
| **Master** | 29,900원 | VIP 매칭, 수익 98%, 마케팅 지원, 고객 직접 피드 | 최상위 전문가(월 50건+) |

### B2B 수익 (Phase 3)

- **보험사 API**: 손해보험 회사에게 누수 사건 처리 API 제공 (월 500,000원)
- **건설사 계약**: 정기점검 견적 패키지 (월 1,000,000원)

## 7. North Star Metric

### 추천 NSM: **월간 활성 전문가(MAE)당 평균 월 거래액**

- 목표: 2,000,000원 (12개월 후, = 거래 20건 × 100,000원 평균)
- = (월 총 거래액) / (월간 활성 전문가)

### Input Metrics (주간 추적)

**Supply Side (전문가)**:
1. 월간 활성 전문가 (MAE) - 목표: 50명 (3개월), 200명 (12개월)
2. 전문가 수락률 - 목표: 90% (매칭된 건의 90%를 수락)
3. 전문가 별점 - 목표: 4.5+ (고객 평가)
4. 전문가 이탈율 - 목표: 월 5% 이하

**Demand Side (고객)**:
1. 월간 신규 누수 신고 - 목표: 300건
2. 매칭 성공률 - 목표: 90% (신고→전문가 매칭)
3. 일 처리 시간 - 목표: 24시간 이내
4. 고객 NPS - 목표: 70+

**Conversion/Monetization**:
1. 에스크로 거래액 - 목표: 월 50,000,000원
2. 평균 주문가액 (AOV) - 목표: 500,000원
3. 거래 수수료 수익 - 목표: 월 2,500,000원
